# Data Visualization Project (Data Engineering + UX)

## Overview
This project is a cross-functional initiative between **Data Engineers** and **UX Designers**.

The objective is to build a **data visualization solution** that enables **data-driven decision-making**.

### Team Responsibilities

**UX Team**
- Studies Agile and Visual Design
- Owns:
  - Agile processes
  - User research
  - Wireframes
  - UI/UX design

**Data Engineering Team**
- Studies data visualization
- Implements technical solutions based on UX designs

---

## Data Selection

The dataset is selected collaboratively based on **interest and relevance**.

### Supported Formats
- CSV
- Excel
- API-based data

### Suggested Data Sources
- Kaggle: https://www.kaggle.com/datasets
- Swedish Open Data: https://www.dataportal.se/datasets
- YH Statistics:
  - https://www.myh.se/yrkeshogskolan/resultat-ansokningsomgangar
  - https://www.myh.se/yrkeshogskolan/resultat-ansokningsomgangar/resultat-for-kurser
- APIs:
  - CoinMarketCap API
  - Trafiklab API
  - Swedish Food Agency Database

You may also use other datasets if all requirements are met.

## Data Engineering Requirements

### Pass (Godkänt)

- Select a dataset
- Use a shared GitHub repository with:
  - Branches
  - Issues
  - Pull Requests

- Each team member performs their own **EDA**

### EDA Tools
- Pandas
- DuckDB

### Power BI Dashboard
- Must include:
  - Filters
  - Relevant KPIs
  - At least:
    - 1 Line Chart
    - 1 Bar Chart
- Must be published

### Data Storytelling
- Create at least **2 visualizations using Matplotlib**

### Delivery
- Present in demos and final presentation

---

### Pass with Distinction (VG)

- Build a **Streamlit dashboard** with:
  - KPIs
  - Filters
  - At least:
    - 1 Line Chart
    - 1 Bar Chart

- Deploy the Streamlit dashboard

### Code Quality
- Well-structured code
- Follows **DRY principles**

### Power BI Enhancements
- Implement **What-if analysis**
- Ensure:
  - Good design practices
  - User-friendly experience

### Data Storytelling
- Strong storytelling principles applied
